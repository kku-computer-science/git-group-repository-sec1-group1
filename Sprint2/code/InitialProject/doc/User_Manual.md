### 2. สร้างประกาศรับสมัครคนเข้าร่วมทำวิจัย
#### วิธีดูข้อมูล  

User - หัวหน้ากลุ่มวิจัย

1. เมื่อเข้าสู่ระบบและเข้ามายังหน้า RESEARCH GROUP แล้ว หัวหน้ากลุ่มวิจัยจะสามารถกดปุ่ม **เพิ่มการประกาศรับสมัคร** ได้

<br>
<br>
